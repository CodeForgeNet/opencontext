# PCSL (Personal Context Sovereignty Layer) Spec v1.0

PCSL is an open protocol that allows users to own, store, and selectively share personal context with AI systems.

## Core Principles
1. **User Ownership**: Context lives on user-controlled infrastructure.
2. **Scoped Access**: AI apps get only what they need (e.g., `preferences`, `skills`).
3. **Auditable**: Every access is logged and transparent to the user.
4. **Portable**: Any AI system can integrate with standard endpoints.
5. **Offline-first**: Works without a mandatory cloud server.

## Repository Structure
- `SPEC.md`: The formal protocol definition.
- `SCHEMA.md`: JSON-LD schema reference for context objects.
- `SECURITY.md`: Threat model and security best practices.
- `examples/`: Reference implementations and context files.
